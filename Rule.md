# プロジェクト概要
楽天アフィリエイトを活用した高単価（3万円以上）特化型の厳選メディア「LifeTech Select (LTS)」のプロジェクトです。

## AIエージェントへの絶対ルール (The LTS Standard)

このプロジェクトを操作するすべてのAIエージェント（CEO）は、作業に着手する前に必ず **プロジェクト直下にある `workflow.md`** を熟読し、そこに定義された経営PDCAおよび記事作成SOPに厳格に従うこと。

- 汎用的なアフィリエイト記事の量産は禁止。
- 必ずウェブ検索を用いた「名機」の特定と、独自の解析テキスト（不都合な真実など）を記述すること。
- 定期的にデータを解析し、サイトを最適化すること。
